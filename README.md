# Pomodoro Timer

Focus timer with work/short-break/long-break modes, an animated progress ring, session tracking, and an end-of-session chime.

## Live Demo

https://srausahowl.github.io/pomodoro-timer/

## Usage

Open `index.html` in any modern browser. Zero dependencies, zero build step.

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Tech

- Vanilla HTML / CSS / JavaScript
- No frameworks, no build tools
